# Development

- Clone repository 

```shell
git clone git@github.com:gustavo-flor/muuusic-bot.git && cd muuusic-bot
```

- Clone `.env.example` to `.env`

```shell
cp .env.example .env
```

- Adjust external tokens on `.env`

- Run muuusic bot locally

```shell
npm run start
```
